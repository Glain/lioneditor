; patch list 样本以及说明

; 所有字符都不区分大小写。

; 这行是注释。以;和#开头的都是注释。只能用半角。
; 另外空行也可以。

; 写入地址必须是8位十六进制地址，不足前面补0
; 写入地址必须是一个2352字节扇区中合法的数据偏移。

; 命令,写入文件, 文件必须和list文件同一目录。
; 命令*写入原始十六进制字符串，当然必须成对。

;下面行写入patch.bin
002F4D38,patch.bin

;下面直接写入字节
002F5098*F5F6F7

;最直观的，把数据写到指定位置的功能
;注意，不重计算校验，不检查写入位置，不进行2048字节分段写入
002F5098@sector1.bin
